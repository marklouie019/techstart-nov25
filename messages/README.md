# Messages Folder

This folder contains individual message files from workshop participants.

## File Naming Convention

Create a file named `your-username.json` (e.g., `johndoe.json`, `jane-smith.json`). Use your unique GitHub username for `your-username.

## JSON File Format

Your JSON file should follow this structure:

```json
{
  "author": "your-username",
  "message": "Your message, quote, or 'I was here' note here!"
}
```
